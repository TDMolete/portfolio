# Tumisang Molete – portfolio

Hey 👋, welcome to the repo for my personal portfolio site. I built it to show who I am, what I’ve been working on, and how I think about building software – one small step at a time.

It’s a static site (just HTML, CSS, and a bit of JavaScript) and it’s hosted on GitHub Pages. If you’re a recruiter or a fellow dev, I hope this gives you a feel for my style and what I care about.

## What’s inside

- A short **about me** – where I’m from, what I studied, and why I build things.
- My **build philosophy** – I’m a big fan of starting with a Minimum Viable Product (MVP) and learning from there.
- The **tech stack** I use – from PHP and Java to VS Code and MySQL.
- Three **featured projects**:
  - **MedTrack** – a lightweight healthcare prototype (PHP/MySQL).
  - **KasiGrowth** – a concept for helping township businesses get online.
  - **E-Threads** – a front‑end MVP for an e‑commerce clothing store.
- What I’m **learning right now** – cloud, security, data, AI agents.
- A clear call‑out that I’m **open to internships and junior roles**.
- My **GitHub stats** (automatically pulled in) and **contact links**.

I also added some **fade‑in animations** and smooth scrolling to make it feel a bit more alive – nothing fancy, just enough to keep it friendly.

## Tech used

- HTML5
- CSS3 (with a warm, custom colour palette)
- Vanilla JavaScript (Intersection Observer for the animations, plus a little smooth scroll)
- Font Awesome icons
- Google Fonts (Plus Jakarta Sans)

## Running it locally

If you want to poke around or use it as a starting point for your own portfolio:

1. Clone the repo:  
   `git clone https://github.com/TDMolete/TDMolete.github.io.git`
2. Open `index.html` in your browser – that’s it. No build step, no dependencies.
